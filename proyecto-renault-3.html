Full-Stack / Soluciones Operativas

Digitalización de Puestos de Trabajo y Gestión de Datos

PHP & MySQL JavaScript Batch Scripting

Desarrollo integral de plataformas web para el control de producción, mantenimiento y batonaje, optimizando el registro de datos y la trazabilidad administrativa.

01. Arquitectura de Datos y Control Operativo

Lideré el diseño y creación desde cero de bases de datos **MySQL** para los módulos de Puesto Fijo, Batonaje y Mantenimiento. Desarrollé interfaces dinámicas que permiten a los operarios registrar tareas y consultar historiales en tiempo real. Implementé un sistema de alertas visuales mediante colores, categorizando la importancia de los registros según sus fechas de vencimiento, garantizando que ninguna tarea crítica quedara desatendida.

Funcionalidad Pro

Exportación automatizada a Excel de historiales completos, facilitando el análisis de datos inmediato para la Jefatura de Unidad.

02. Panel de Administración y Seguridad

Sistema de Auditoría (Audit Log)

Para garantizar la integridad de la información, desarrollé un panel administrativo con permisos avanzados. A diferencia del historial general, este módulo registraba con identificación unívoca quién, cuándo y qué dato había sido modificado o eliminado, proporcionando una trazabilidad total para la gestión de administradores.

Integración Visual Corporativa

Toda la interfaz fue diseñada siguiendo estrictamente la identidad visual de Renault. Además, elaboré documentación técnica y presentaciones (PPTX) para la formación de operarios y explicación del flujo de datos ante los responsables del área.

03. Soporte de Infraestructura e IT

Mi labor fue más allá del desarrollo web, actuando como soporte técnico integral. Me encargué de la instalación física de ordenadores en línea, la reconfiguración de puestos de trabajo y la modificación de archivos .bat (Batch) para la automatización de procesos locales. Esta visión me permitió asegurar que el software desarrollado funcionara perfectamente sobre el hardware y la red de la planta.
