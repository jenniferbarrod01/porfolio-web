Portal Cultural CyL

Laravel React Tailwind CSS API Integration

Solución interactiva para centralizar el acceso a la agenda cultural de Castilla y León mediante el consumo de datos abiertos oficiales.

01. Idea de proyecto

El objetivo principal fue transformar un flujo masivo de datos crudos (procedentes de la API de la Junta de CyL) en una experiencia de usuario fluida, visual y útil. El reto técnico residía en la normalización de datos asíncronos y su representación en un entorno interactivo sin sacrificar el rendimiento de la aplicación.

  • Consumo de API REST pública
  • Gestión de estados complejos
  • Geolocalización dinámica

02. Implementación Técnica

Arquitectura Desacoplada

Uso de Laravel como motor de backend para gestionar la lógica de negocio y el enrutamiento, mientras que el frontend fue desarrollado íntegramente con React, permitiendo una navegación tipo SPA (Single Page Application) que elimina los tiempos de recarga.

Geolocalización e Interactividad

Implementación de un mapa interactivo que permite al usuario explorar eventos por proximidad geográfica. Los marcadores se actualizan dinámicamente según los filtros aplicados, vinculando la ubicación del evento con su ficha técnica detallada.

Sistema de Filtrado Avanzado

Desarrollo de un motor de búsqueda que permite filtrar por provincia, categoría cultural, fechas, precio, etc. facilitando la búsqueda personalizada entre cientos de registros actualizados en tiempo real.
